# Elderly-Assistive-Exoskeleton
Design and simulation of a wearable robotic exoskeleton for elderly assistance
A personal project of mine to develop an affordable robotic exoskeleton for the elderly and even for other purposes.

## Project Overview
A wearable robotic exoskeleton designed to assist elderly individuals with mobility.

## Design Details
* **Software used:** Autodesk Fusion 360
* **Key Features:** Ergonomic joints, lightweight alloy frame, and gait-mimicking actuation points.
